# 🚀 Быстрый старт BioLink

## Развертывание на Railway за 3 шага

### Шаг 1: Загрузить на GitHub
```bash
git init
git add .
git commit -m "BioLink platform"
git remote add origin <your-repo-url>
git push -u origin main
```

### Шаг 2: Подключить к Railway
1. Зайти на [railway.app](https://railway.app)
2. Нажать **New Project**
3. Выбрать **Deploy from GitHub**
4. Выбрать ваш репозиторий
5. Railway автоматически развернет

### Шаг 3: Открыть сайт
- Ваш сайт будет доступен по адресу Railway
- Пример: `your-project.up.railway.app`

## 🆘 Помощь

**Сборка не работает?**
```bash
npm install
npm run build
```

**Сервер не запускается?**
```bash
node server.js
# Проверьте порт 8080
```

**Данные потерялись?**
- Данные в localStorage
- Очистка кеша = потеря данных
- Для продакшена нужна БД
