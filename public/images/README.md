# Логотип BioLink

## Инструкция

Поместите ваш файл **logo.png** в эту папку (`public/images/`).

### Требования к файлу:
- **Название:** `logo.png`
- **Формат:** PNG (с прозрачностью)
- **Размер:** Рекомендуется 64x64px или больше
- **Качество:** Высокое разрешение для четкого отображения

### Где используется:
1. **Favicon** — иконка в браузерной вкладке
2. **Главная страница** — логотип слева вверху в навбаре

### После добавления:
Логотип автоматически появится в этих местах.

### Структура:
```
public/
└── images/
    └── logo.png  ← Положите файл сюда
```

### Примечание:
Если файл `logo.png` отсутствует, браузер может показать ошибку 404 в консоли.
