## 🚀 Как работает экран приветствия

### Когда показывается:
- Только если у пользователя загружена музыка
- При первом открытии страницы

### Внешний вид:
```
┌─────────────────────────────┐
│   Blur + Gradient фон       │
│                             │
│   🎵 Animated Icon          │
│   Добро пожаловать          │
│   На странице Username      │
│                             │
│   [🎵 Открыть]             │
│   [✕  Закрыть]             │
│                             │
│   Мелкий текст о музыке     │
└─────────────────────────────┘
```

### Логика:
1. **Открыть:**
   - Скрывает экран приветствия
   - Запускает музыку (`audio.play()`)
   - Показывает основную страницу

2. **Закрыть:**
   - Пытается закрыть вкладку (`window.close()`)
   - Если не получается — возвращает на главную
