## 🚀 Деплой на Railway

### Шаг 1: GitHub
```bash
git init
git add .
git commit -m "BioLink platform ready"
git remote add origin <your-repo>
git push -u origin main
```

### Шаг 2: Railway
1. Зайти на railway.app
2. New Project → Deploy from GitHub
3. Выбрать репозиторий
4. Railway автоматически развернет проект

### Шаг 3: Настройка
1. В Railway откройте Settings
2. В Networking добавьте домен
3. Готово! Сайт работает
