# BioLink — Бесплатная платформа для bio-ссылок

Современная платформа для создания красивых bio-страниц с поддержкой всех популярных социальных сетей.

## Возможности

✨ **Полностью бесплатно** — все функции доступны всем пользователям  
🎨 **Кастомизация** — настройка цветов, фона и музыки  
🔗 **14+ платформ** — GitHub, Twitter, Instagram, YouTube, Telegram, Discord и многое другое  
📸 **Аватары** — загрузка JPG, PNG (до 3MB), GIF для создателя (до 10MB)  
🎵 **Фоновая музыка** — загрузите MP3, OGG или WAV (до 10MB)  
🖼️ **Кастомный фон** — загрузите изображение JPG, PNG или GIF (до 5MB)  
📁 **Загрузка файлов** — файлы конвертируются в base64, не требуется хостинг  
🎭 **Демо режим** — посмотрите пример перед регистрацией  
🔐 **Безопасность** — капча, защита от ботов  
📊 **Админ панель** — полный контроль для администраторов  
🌐 **Glass Design** — современный стеклянный дизайн

## Развертывание на Railway

### Автоматическое развертывание

1. Залогиньтесь в [Railway](https://railway.app)
2. Создайте новый проект
3. Подключите этот репозиторий
4. Railway автоматически обнаружит `railway.json` и выполнит сборку

### Переменные окружения

Не требуются! Все работает из коробки.

### Порт

Приложение слушает на порту `8080` (настраивается через `PORT` env variable).

### Сборка и запуск

```bash
# Сборка
npm install && npm run build

# Запуск
node server.js
```

## Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```

## Формат URL

Публичные страницы доступны по адресу:
- `/@username`
- `/bio/username`
- `/u/username`

## Хранение данных

Данные хранятся в `localStorage` браузера:
- Профили пользователей
- Ссылки и настройки
- Изображения фона (base64)
- Музыкальные файлы (base64)

**Важно:** 
- localStorage имеет лимит ~5-10MB на домен
- Для больших файлов может потребоваться внешний хостинг
- Для продакшена рекомендуется добавить backend с базой данных и файловым хранилищем
